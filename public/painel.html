📋 Diretrizes da Comunidade

Leia as regras de uso da Crina.

Ao utilizar a Kryna, você concorda em respeitar todos os membros das comunidades onde o bot está presente.

1. Uso adequado: Não utilize os comandos para assediar, ofender ou prejudicar outros membros.

2. Moderação: Os recursos de moderação devem ser usados de forma justa e responsável por administradores.

3. API: Não compartilhe sua chave de API e não utilize a API para atividades automatizadas abusivas.

4. Conteúdo: Não configure a Kryna para enviar ou moderar conteúdo ilegal, ofensivo ou impróprio.

5. Privacidade: Respeite a privacidade dos usuários ao configurar logs e registros de eventos.
